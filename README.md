Travel Crafters is a comprehensive and feature-rich Tours & Travels web application designed to provide users with a smooth and convenient travel booking experience. The platform includes a powerful admin panel for managing destinations, travel packages, bookings, and payments efficiently.

---

## Features

### User Features

- Browse travel destinations and tour packages
- View detailed package information including itinerary and pricing
- Check real-time seat availability
- Book tour packages online
- Secure payment processing through integrated payment gateway
- View booking history and booking details
- User profile management
- Password reset functionality via email
- Submit feedback and inquiries

---

### Admin Features

- Secure admin authentication
- Manage users
- Add, update, and delete destinations
- Create and manage travel packages
- Manage package itineraries, inclusions, exclusions, and availability
- View and manage bookings
- Track and manage payment transactions
- View user feedback and contact inquiries
- Admin dashboard for overall system monitoring

---

### Additional Features

- Secure payment gateway integration
- Password reset system with token-based verification
- Session-based user authentication
- Responsive UI using Bootstrap
- Real-time seat availability check using AJAX
- Clean and user-friendly interface
